# ctf-scripts
*by Matteo Iacovantuono*

## Overview
A collection of custom scripts I have developed to exploit vulnerabilities encountered in TryHackMe labs and CTF environments.

## Scripts

### thm-domino
`jwt-exploit.py` - A Python tool that exploits a JWT signature verification vulnerability. Forges an admin token to access a private API and read files from the server.
